# halcyon
###### a 200x inspired chat application for the modern age

## about
halcyon is a project born out of a desire for a couple of things. i wanted something private and high quality, but which
people would be happy to switch to from more popular alternatives which shall not be mentioned, but i'm sure you can
guess.

## contributing
at the stage i'm at right now, i want to keep it solo. there may, however, be issues with `help-wanted` tags, and if you
see one, feel free to give it a go!
